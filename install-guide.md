# 需求環境
  npm ^5.5.1
  node ^6.11.2
  yarn ^1.2.1

# 可運行平台OS(經測試)
  1. Windows
  2. OSX

# 安裝步驟
  1. Install node_modules
     ```
     yarn install
     ```

# Build & Test
  ## Dev mode
  ```
  yarn dev
  ```
  ## Linting
  ```
  yarn lint
  ```

  ## Production

  ```
  yarn prod
  ```
